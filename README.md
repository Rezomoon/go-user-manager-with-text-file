🇺🇸 English 
Go User Manager (CLI Practice Project)

A simple command-line project written in Go for practicing:

Reading user input

Working with slices and maps

File operations (saving data in a text file)

Adding, listing, and deleting users

📌 Features

Add new user (username & password)

Show all users

Delete a user

Save users in a .txt file

Simple CLI menu

📦 How to Clone & Run
git clone https://github.com/Rezomoon/go-user-manager-with-text-file.git
cd go-user-manager
go run main.go

🎯 Purpose

This project is NOT for real authentication.
It's only for learning Go basics such as file handling, CLI, and data structures
